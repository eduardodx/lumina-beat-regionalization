# Notebooks

This directory holds exploratory notebooks and small audit notebooks.

## Current Inventory

- `data.ipynb`: general notebook used for ad hoc inspection

## ClinVar Organization

ClinVar-specific exploration should live under `notebooks/clinvar/` and stay split by purpose:

- filters and inclusion/exclusion decisions
- metadata/schema inventory
- split and label audits

Any notebook that affects the ClinVar dataset definition should be mirrored in
[ClinVar Dataset Curation](../docs/eval/clinvar-dataset.md).
